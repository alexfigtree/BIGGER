How to play: Use your arrow keys to move the tiles. When two tiles of the same size touch, they merge into one and become BIGGER!

About: BIGGER is game is based on Gabriele Cirulli's 2048. It was made with young children in mind, to help them learn the conceptual differences between big and small. No math necessary!

BIGGER is a game created by Alex Higuera, based on the original 2048 game created by Gabriele Cirulli., which is based on 1024 by Veewo Studio and conceptually similar to Threes by Asher Vollmer.
